
# N-Blog

## 网站的技术结构
* nodejs(express) + mongodb

## 运行说明
  1. npm install -g node-dev 安装node-dev,方便开发
  2. 安装mongodb，并启动
  3. npm start启动项目

## 网站的几个基础功能：

  * 博客功能
    * 后台文章发布
    * 前台文章倒叙展示，limit 15
    * 前台文章分类：类别，每个类别下的列表
